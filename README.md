A simple and responsive QR Code Generator built using HTML, CSS, and JavaScript.
This project allows users to generate QR codes instantly by entering any text or URL. It uses a QR Code API to dynamically create QR images.

🚀 Features

Generate QR code for any text or URL

Instant QR code preview

Simple and clean user interface

Responsive design (works on mobile & desktop)

Uses external API for fast QR generation

🛠️ Technologies Used

HTML – Structure of the webpage

CSS – Styling and layout

JavaScript – Functionality and API integration

QR Code API – To generate QR images dynamically
